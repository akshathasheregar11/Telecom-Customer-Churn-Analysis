# 📊 **Telecom Customer Churn Analysis**



## **SQL + Power BI | Customer Retention Analytics Project**



###### **🔎 Overview**



A data analytics project analyzing customer churn in a telecom business using SQL Server and Power BI.



The goal was to identify churn patterns, segment customers based on risk, and build an interactive dashboard to support retention-focused business decisions.



###### **🎯 Objective**

* Understand key drivers of customer churn
* Identify high-risk customer segments
* Analyze churn across contract types and services
* Build a dashboard for business stakeholders to monitor churn trends



###### **🛠️ Tools \& Technologies**

* SQL Server
* Power BI
* Git \& GitHub



###### **⚙️ Approach**

* Performed data validation and quality checks using SQL
* Built reusable SQL views for structured analysis

**Analyzed churn across:**

* Contract types
* Internet services
* Tenure and monthly charges
* Developed customer segmentation logic using SQL
* Identified high-risk customers using conditional logic
* Built an interactive Power BI dashboard for visualization



###### **🧠 Key SQL Techniques**

* Common Table Expressions (CTEs)
* CASE statements
* Aggregate functions
* GROUP BY
* Views for modular querying



###### **📊 Power BI Dashboard**



An interactive dashboard designed for business users to explore churn patterns and customer risk profiles.



**Key Views:**

* KPI Summary (Total Customers, Churn Rate, Revenue Impact)
* Churn by Contract Type
* Churn by Internet Service
* Customer Segmentation
* High-Risk Customer Identification



📌 Includes filters for dynamic exploration across customer groups.




###### **💼 Business Solution**



Based on the churn analysis, the following actions can be implemented to reduce customer attrition:



* Target customers on month-to-month contracts with upgrade offers to long-term plans
* Launch early-life engagement programs for customers in the first few months of onboarding
* Create retention campaigns for high-risk customers identified through segmentation
* Monitor customers with high monthly charges and low tenure for proactive intervention
* Offer personalized discounts or benefits for customers showing early signs of churn
* Improve service experience for customers using higher-churn internet service types



###### **📈 Key Insights**

* Month-to-month customers show the highest churn rate
* Customers with shorter tenure are significantly more likely to churn
* Higher monthly charges are strongly associated with churn risk
* Customer segmentation helps identify priority retention groups
